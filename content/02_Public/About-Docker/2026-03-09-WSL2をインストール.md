- [[WSL2]]が入っているか確認。入っているならバージョンが出る
```
wsl --status
```

- WSL2をインストール
```
wsl.exe --install
```
- ユーザ名を決める: 英数字。日本語や空白厳禁。
- パスワードを決める: しっかり覚えておく。パスワードを打ち込む際はUIに出てこないが、落ち着いて打ち込む。

- インストール後、再起動する

- WSL2を確認
```
wsl --status
```
- 【出力】
- 既定のディストリビューション: [[Ubuntu]]
- 既定のバージョン: 2
- WSL1 は、現在のマシン構成ではサポートされていません。
- WSL1 を使用するには、"Linux 用 Windows サブシステム" オプション コンポーネントを有効にしてく ださい。
